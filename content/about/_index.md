description: "Overview of the starter kit and how to adapt it for your own app."
---

This starter kit is meant to help you stand up a Hugo site on Azure Static Web Apps with:

- Microsoft login through EasyAuth
- Azure Functions-based auth checks
- role-aware navigation and feature gating
- a simple admin surface for role management

Use it as a foundation for internal tools, partner portals, or public sites with authenticated feature areas.
