What this starter demonstrates

EasyAuth login flow

Login and logout are handled by Azure Static Web Apps. The browser checks auth state through /.auth/me.

Role-aware UI

Navigation and example project cards react to the current role returned by /api/check-role.

Backend authorization

Azure Functions read x-ms-client-principal, resolve roles from blob storage, and enforce access for privileged paths.

Secure content delivery

All restricted page content is served through authorized Azure Function endpoints from private blob storage. No sensitive content is baked into static HTML.
