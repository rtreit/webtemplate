About this starter kit

This starter kit helps you stand up a Hugo site on Azure Static Web Apps with:

  • Microsoft login through EasyAuth
  • Azure Functions-based authorization checks
  • Domain-gated content served from private blob storage
  • Role-aware navigation and feature gating
  • A simple admin surface for role management

Use it as a foundation for internal tools, partner portals, or authenticated web applications.

Security architecture

This page and all restricted content on this site are served through an authenticated Azure Function that reads x-ms-client-principal and verifies the signed-in user's email domain before returning any content. The static HTML you see in the browser is just a loading shell. The real content only arrives after server-side authorization passes.

See the Auth Architecture page for a detailed walkthrough with diagrams.
